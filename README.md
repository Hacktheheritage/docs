# Kyrgyz Heritage Platform

## Overview
This repository contains documentation for a digital platform that integrates:
- Sacred sites
- Petroglyph heritage
- Traditional Kyrgyz calendar

The goal is to preserve cultural knowledge while providing interactive, educational, and engaging features.

## Contents
- `README.md` — comprehensive project README and SRS summary
- `frontend.md` — frontend design and feature plan
- `backend.md` — backend API design, architecture, and integration plan

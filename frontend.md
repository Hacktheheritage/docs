# Frontend Documentation

## Purpose
The frontend is the user-facing interface for the Kyrgyz Heritage Platform. It provides navigation, content presentation, chatbot interaction, and quiz engagement.

## Main Components

### Homepage
- Shows an overview of all topics
- Allows navigation to:
  - Sacred Places
  - Petroglyphs
  - Calendar
- Includes a short introduction to the platform
- Features a call-to-action for the chatbot and quizzes

### Sacred Places Module
- Displays a list or map of sacred places
- Shows descriptions and cultural meaning
- Includes guidance on practices and heritage etiquette

### Petroglyph Module
- Displays images of petroglyphs
- Provides historical explanations
- Includes interpretation support or storytelling

### Calendar Module
- Shows the traditional Kyrgyz calendar representation
- Compares the traditional calendar to the modern calendar
- Provides educational explanations and cultural context

### AI Chatbot
- Visible as a floating assistant or popup
- Prompts users with "Have questions?"
- Handles Q&A about sacred sites, petroglyphs, and calendar topics
- Uses a simple text input and chat thread layout

### Quiz Experience
- Displays multiple choice questions after each section
- Collects quiz answers and calculates score
- Offers a results summary and optional leaderboard

## UI Requirements
- Clean navigation bar
- Responsive design for mobile and desktop
- Accessible fonts and touch-friendly elements
- Simple layout, not overcomplicated

## User Flow
1. User lands on homepage
2. User selects a domain module
3. User reads cultural content
4. User engages with the chatbot or quiz
5. User reviews results and may explore another module
